Question-02:What Does Javascript Event Loop Do?

We all know,Javascript is a single threaded programming Language.It means that there is a main thread where Javascript code is run.Javascript runs runs code one line at a a time and rhere is no possibility of running code in parallel. Here Javascript allocates memory in Heap Memory and Stack memory.In Stack Memory Functions are stacked. The function of Event loop is something that pulls out of Event Queue and places it in the Function Execution Stack when the function execution stack is empty.The event loop is constantly running and monitors the Call Back Queue and Call Stack.It is an important part which helps Javascript to run its programs smoothly.Event Loop maintains a relation between Event Queue and Call Stack

Question-03:Difference Between Local Storage and Session Storage

Both Local Storage and Session Storage are ways in which the browsers use to store values in key value pais for a particular session domain using Web Api.The values stored in both the storage may be strings.The main difference between Local Storage And Session Storage is in the Expiration sessions of the datas .In Local Storage,data persists across browser sessions. which means if the browser is closed ,this storage saves the Data.There is no expiration of Data. On the other hand, Incase forSession Storage the stored data is cleared as soon as the page ends or the browser is closed.That means,ehen a new window or new tab is opened,this creates a new session and new storage for the user..
